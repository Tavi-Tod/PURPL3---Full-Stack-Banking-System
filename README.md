# PURPL3---Full-Stack-Banking-System
PURPL3 - Sistem Bancar Full Stack. O aplicație web modernă cu design Glassmorphism, dezvoltată folosind Node.js (Backend) și MySQL (Database). Include funcționalități de Înregistrare cu pachete bancare, Dashboard pentru transferuri în timp real, Istoric tranzacții și un Panou de Admin securizat pentru gestionarea utilizatorilor.
